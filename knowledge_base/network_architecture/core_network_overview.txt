===========================
CORE NETWORK TOPOLOGY
===========================

The enterprise network is divided into three primary zones:
1. DMZ (Demilitarized Zone) - External-facing services
2. Internal Network - Core business applications
3. Restricted Zone - Sensitive financial and compliance systems

All traffic between zones passes through the Central Firewall Cluster (CFC).

===========================
REDUNDANCY AND FAILOVER
===========================

Active-passive redundancy for all core switches. Failover time under 30 seconds.
Primary data center: DC-WEST. Failover: DC-EAST. Connected via dual 10Gbps MPLS links.

===========================
MONITORING
===========================

All network devices monitored via Zabbix on 10.10.50.20.
SNMP traps forwarded to central SIEM at 10.10.50.100.
