BLOCKED PORTS
===========================

Port 23 (Telnet) — Blocked. Insecure.

Port 3389 (RDP) — Blocked. Use jump server via SSH.

===========================
REQUESTING A NEW PORT
===========================

Submit CAB request via ServiceNow portal (ticket type: Network Change).
Required fields: business justification, source/dest IPs, port, protocol, duration.
Approval: Network Security Lead + Application Owner.
SLA: 5 business days.
